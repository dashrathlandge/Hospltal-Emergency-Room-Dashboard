# Hospltal-Emergency-Room-Dashboard
Built an interactive Hospital Emergency Dashboard in Microsoft Excel using Power Query, Pivot Tables, Pivot Charts, KPI Cards, and Slicers. Analyzed patient admissions, wait times, satisfaction scores, department referrals, age groups, and gender distribution to create actionable healthcare insights.

An interactive **Hospital Emergency Dashboard** built in **Microsoft Excel** to analyze patient admissions, waiting times, patient satisfaction, department referrals, and demographic trends. This project demonstrates an end-to-end data analysis workflow, from data cleaning to dashboard creation.

## 📌 Project Overview

This project transforms raw hospital emergency data into an interactive dashboard that helps monitor hospital performance and supports data-driven decision-making.

The dashboard provides insights into:

* Patient admissions
* Average wait time
* Patient satisfaction
* Attendance status
* Department referrals
* Gender distribution
* Age group analysis
* Monthly patient trends
